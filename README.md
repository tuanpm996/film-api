Film API
========

## Prerequisite

> - Python (version > 3.0)

## Installation

```bash
pip install flask
```

## How to run

To run the project, run the following command in Git Bash.

```bash
export FLASK_APP=index.py
flask run
```

Or run:

```bash
bash init.sh
```
