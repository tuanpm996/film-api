Film API
========

## Prerequisite

> - Python (version > 3.0)

## Installation

```bash
pip install flask
pip install -U flask-cors
```

## Documentation

> - [Flask](http://flask.pocoo.org)
>
> - [Flask CORS](http://flask-cors.readthedocs.io/en/latest/)

## How to run

To run the project, run the following command in Git Bash:

```bash
bash init.sh
```

## API

> - [localhost:5000/films]()
