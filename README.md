Film API
========

## Prerequisite

> - Python (version > 3.0)

## Installation

```bash
pip install flask
```

## Documentation

> - [Flask](http://flask.pocoo.org)
>
> - [Flask OpenID](https://pythonhosted.org/Flask-OpenID/)

## How to run

To run the project, run the following command in Git Bash.

```bash
export FLASK_APP=index.py
flask run
```

Or for short:

```bash
bash init.sh
```
