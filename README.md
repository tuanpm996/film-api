Film API
========

## Prerequisite

> - Python (version > 3.0)

## Installation

```bash
pip install flask
pip install flask-restful
```

## How to run

To run the project, run the following command in Git Bash.

```bash
export FLASK_APP=api.py
flask run
```

> - [localhost:5000/films]()
